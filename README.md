## 📋 Инструкция по установке

### 1. Добавьте видеофайлы

Поместите следующие видео в папку `videos/`:

| Файл | Описание | Рекомендуемый размер |
|------|----------|---------------------|
| `background.mp4` | Фоновое видео (зацикленное) | 1920x1080, до 10MB |
| `clip1.mp4` | Клип "Эпичный момент" | 400x400, до 3MB |
| `clip2.mp4` | Клип "Клатч" | 400x400, до 3MB |
| `clip3.mp4` | Клип "Фан" | 400x400, до 3MB |
| `clip4.mp4` | Клип "Хайлайт" | 400x400, до 3MB |
| `clip5.mp4` | Клип "Вин" | 400x400, до 3MB |

### 2. Настройте ссылки на соцсети

Откройте `index.html` и замените ссылки в секции `.social-links`:

```html
<a href="https://twitch.tv/ВАШ_НИК" ...>
<a href="https://youtube.com/@ВАШ_КАНАЛ" ...>
<a href="https://discord.gg/ВАШ_СЕРВЕР" ...>
<a href="https://t.me/ВАШ_КАНАЛ" ...>
<a href="https://tiktok.com/@ВАШ_НИК" ...>
```

### 3. Запуск сайта

Просто откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Если установлен Python
python -m http.server 8000

# Если установлен Node.js
npx serve
```

## 🔧 Требования к видео

### Фоновое видео
- Формат: MP4 (H.264)
- Разрешение: 1920x1080 или выше
- Длительность: 10-30 секунд (зациклено)
- Размер: до 10 MB

### Клипы
- Формат: MP4 (H.264)
- Разрешение: квадратное (400x400 или 500x500)
- Длительность: 5-15 секунд (зациклено)
- Размер: до 3 MB каждый
